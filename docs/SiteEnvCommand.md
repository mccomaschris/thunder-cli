# thundr site:env

## Description

View or edit the .env file on the remote site.

## Usage

```bash
thundr site:env
```

## What it does

- Supports interactive editing, adding, and removing variables.
- Backs up `.env` before writing changes.

## Example

```bash
thundr site:env
```

## Notes

Requires SSH access to the server and working site path.

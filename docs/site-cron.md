# `site:cron` Command

## 📝 Description

Toggle the cron job to run the Laravel scheduler for Artisan commands

---

## 🚀 Usage

```bash
php thundr site:cron
```





---

## ✅ Example

```bash
php thundr site:cron
```

---

## 🧠 Details

This command is implemented in the `SiteCronCommand` class.
